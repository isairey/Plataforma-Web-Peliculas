# 🌌 Acerca del proyecto

**Movie Streaming Application** es una plataforma web desarrollada para ofrecer contenido multimedia de manera moderna e intuitiva.

La aplicación permite explorar películas y series, visualizar información multimedia y disfrutar de una experiencia de streaming fluida desde cualquier dispositivo.

El proyecto fue creado para practicar:

- 🎬 Desarrollo de plataformas multimedia
- ⚛️ Interfaces modernas con React
- 🌐 Consumo de APIs
- 📱 Responsive Design
- 🚀 Arquitecturas Full Stack
- 🎨 Diseño UI/UX

# ⚡ Instalación

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/Plataforma-Web-Peliculas.git
```

---

## 2️⃣ Instalar dependencias

```bash
npm install
```

---

## 3️⃣ Ejecutar proyecto

```bash
npm start
```

---

## 4️⃣ Abrir aplicación

```bash
http://localhost:3000
```
